Grocery Store Clerk PLU Code Training

Welcome to the PLU trainer application. Use the practice page to practice PLU codes to help memorize them and improve your accuracy as a clerk at most grocery stores.

Please note that store-specific PLU codes may not be included in this application- treat this as a helper to learn generic codes.

To start, check out the Cheat Sheet, or start one of the following practice sessions:

  • PLU Code Practice - Practice identifying items based on their PLU codes.
  • PLU Image Practice - Practice identifying PLU codes based on item images. (Coming Soon)
  • PLU Matching Game - Match items to their correct PLU codes in a fun game format. (Coming Soon)

About This Application

This project is Open Source and was created by Miles Hilliard. For more information about the developer, visit the website.

The database used in this application is from The International Federation for Produce Standards (IFPS)
